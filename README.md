## Setup

### Install dependencies

```sh
yarn
```

### Start Development Server

```sh
yarn dev
```

### Building and Running for Production

1. Generate a production build

   ```sh
   yarn build
   ```

1. Preview the site

   ```sh
   yarn serve
   ```
